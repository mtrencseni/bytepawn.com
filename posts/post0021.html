Google Docs Preview Firefox Addon google-docs-preview-firefox-addon 2009/09/10

Download the Google Docs Preview Firefox Addon here. Google has exported their very cool web-based PDF viewer. You can use it by telling it what PDF file to open like this: 
http://docs.google.com/gview?url=PDF_URL
MARKER For example, here is the Keyspace whitepaper shown in Google Docs Preview: http://docs.google.com/gview?url=http://scalien.com/pdf/Keyspace.pdf You can also embed it in an IFRAME like this:
<iframe src="http://docs.google.com/gview?embedded=true&url=PDF_URL"
style="width:800px; height:600px;" frameborder="0"></iframe>
